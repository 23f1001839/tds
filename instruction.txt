Must have python installed in pc
Open command prompt in this folder location

after opening this folder you can create virtual environment by below command with vscode terminal (recommended):
python3 -m venv .env

source .env/bin/activate (for linux)
.env\Scripts\activate (for windows)

(Install all dependencies, Only one time by below code)
pip install -r requirenments.txt 

Finally run:
python3 main.py
